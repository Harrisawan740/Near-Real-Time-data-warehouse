
_*_*_*_*_*_*_*_*_*_*_*_*_* HOW TO RUN THE PROJECT   *_*_*_*_*_*_*_*_*_*_*_*_*_*_*_*_


1- Run the Transactional_MasterData Generator.sql in MYSQLWORKBENCH 8.0.31 

2- Run the createDW.sql in same workbench

3- Import the Maven project in Eclipse

4- Run the project as one JAVA PROJECT

5- Enter credentials of the database i.e. root,password,database

6- Here we go

7- Run the Queries.sql to get the results

Dependencies:

JAVA SDK 
MAVEN
